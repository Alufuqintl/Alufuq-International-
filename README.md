# Alufuq International Website

A modern, premium, responsive website for a Dubai-based food trading company.

## Structure

- `index.html`: Homepage with Hero, About, Strengths, Categories.
- `about.html`: Company overview, Mission, Vision.
- `products.html`: Detailed product categories (Dairy, Bakery, Grocery, etc.).
- `horeca.html`: Solutions for Hotels, Restaurants, and Catering.
- `export.html`: International shipping and logistics services.
- `contact.html`: Contact form and details.

## Customization

### Images
The `assets/` folder is currently empty as the image generation service was unavailable. Please add your own high-quality images:
1.  `hero-bg.jpg` (For Homepage)
2.  `about-us.jpg` (For About Section)
3.  `products-bg.jpg` (For Products Header)
4.  `horeca-bg.jpg` (For HORECA Header)
5.  `export-bg.jpg` (For Export Header)
6.  `contact-bg.jpg` (For Contact Header)

### Colors
You can adjust the brand colors in `styles.css` under `:root`:
- `--primary-color`: Dark Green (#0F3D3E)
- `--secondary-color`: Gold (#D4AF37)

### Content
Update the text content in each HTML file to reflect the specific details of Alufuq International.

## Local Development
To view the site, simply open `index.html` in your browser.
